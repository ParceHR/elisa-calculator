# ELISA Row Calculator — PWA

A mobile web app that installs on iOS and Android home screens with no App Store needed.

## Files to upload to GitHub

Upload all 5 files:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

---

## Step-by-step setup (phone only, ~15 minutes)

### 1. Create a GitHub account
Go to github.com → Sign up (free)

### 2. Create a new repository
- Tap the + icon → New repository
- Name it: `elisa-calculator`
- Set to **Public**
- Tap "Create repository"

### 3. Upload the files
- Open your new repo
- Tap "uploading an existing file" (or Add file → Upload files)
- Upload all 5 files from this zip
- Tap "Commit changes"

### 4. Deploy on Vercel
- Go to vercel.com → Sign up with GitHub (free)
- Tap "Add New Project"
- Select your `elisa-calculator` repo
- Leave all settings as default
- Tap "Deploy"

### 5. Get your link
Vercel gives you a URL like:
`https://elisa-calculator.vercel.app`

Share this link with your colleagues.

## How colleagues install it

**iPhone:**
1. Open the link in Safari (must be Safari, not Chrome)
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

**Android:**
1. Open the link in Chrome
2. Tap the three dots menu
3. Tap "Add to Home screen"
4. Tap "Add"

The app works offline after first load.
